Windows System Monitor

A concise Windows-native monitoring tool that surfaces process and resource behavior in real time using low-level system APIs.

What it demonstrates

The project focuses on fundamentals of operating systems and systems programming. It provides insight into CPU usage, memory consumption, and active ports by interacting with the Windows environment directly through the Win32 API.

Highlights

  • Tracks running processes and system activity.
  • Shows CPU and memory usage trends in a simple interface.
  • Provides a practical introduction to low-level API usage and system monitoring.
