Dump Surveillance AI

An AI-powered waste-management platform that detects illegal dumping through surveillance, strengthens reporting workflows, and turns evidence into actionable insight for authorities.

What it does

The system monitors CCTV data, identifies suspicious dumping activity with YOLO-based object detection, and extracts vehicle number plates using OCR for evidence collection. It also allows citizens to submit reports with images, location, and timestamps, creating a broader picture of illegal disposal in an area.

Core features

  • Real-time detection of dumping behavior from video input.
  • Automated OCR-based plate recognition and incident evidence linking.
  • Interactive hotspot mapping using Leaflet to visualize problem areas.
  • Citizen reporting flow for media uploads and incident descriptions.
  • AI chatbot guidance for responsible waste disposal practices.
